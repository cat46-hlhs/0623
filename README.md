# 俄羅斯方塊 - Web Tetris 遊戲

一個使用 HTML5 Canvas 開發的網頁版俄羅斯方塊遊戲。

## 功能特性

✨ **完整的俄羅斯方塊遊戲機制**
- 7 種不同的方塊形狀（I、O、T、S、Z、J、L）
- 方塊旋轉功能
- 行消除和得分系統
- 難度等級遞增

🎮 **遊戲控制**
- ⬅️ ➡️：左右移動方塊
- ⬇️：加速下落
- ⬆️：旋轉方塊
- 空白鍵：立即落下
- P：暫停/繼續遊戲

🎨 **視覺設計**
- 炫彩霓虹風格界面
- 流暢的動畫效果
- 實時顯示下一個方塊
- 分數、等級、消除行數統計

📱 **響應式設計**
- 支援桌面和移動設備
- 自適應布局

## 使用方法

1. 打開 `index.html` 文件
2. 點擊「開始遊戲」按鈕開始
3. 使用鍵盤控制方塊
4. 消除完整的行數以獲得分數
5. 點擊「重新開始」按鈕重新開始遊戲

## 得分規則

- 消除 1 行：100 分 × 等級
- 消除 2 行：300 分 × 等級
- 消除 3 行：500 分 × 等級
- 消除 4 行：800 分 × 等級

每消除 10 行，等級增加 1，遊戲速度加快。

## 文件結構

- `index.html` - 遊戲 HTML 結構
- `style.css` - 遊戲樣式和動畫
- `tetris.js` - 遊戲邏輯實現

## 技術棧

- HTML5 Canvas
- CSS3
- Vanilla JavaScript (ES6+)
